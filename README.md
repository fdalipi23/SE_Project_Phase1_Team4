Problem Statement

Families often struggle to receive updates during labor and delivery when they are not present at the hospital. This can create stress, confusion, and uncertainty during an important moment. Most existing systems focus only on basic pregnancy tracking and do not include real-time labor monitoring. Therefore, a system that focuses on operation tracking during labor, while also including pregnancy tracking features, is needed.

Proposed Solution

The proposed solution is a web-based Operation Tracking and Pregnancy Support System designed to improve communication, organization, and support during pregnancy and especially during labor and delivery. The main purpose of the system is to provide family members with real-time updates during childbirth so they can stay informed even when they are not physically present at the hospital. In many situations, family members are forced to wait for long periods without knowing the condition of the mother, the stage of labor, or whether the baby has been delivered safely. This creates fear, stress, confusion, and emotional pressure. The system aims to solve this problem by creating a platform where important information can be shared quickly, clearly, and securely.

The main focus of the system is the operation tracking environment during labor. This feature allows authorized family members to receive updates about the labor and delivery process in real time. Different stages of childbirth can be displayed, such as hospital admission, preparation for labor, active labor, operation status, baby delivery, and recovery period. Family members can follow the process through a dashboard that shows the current stage and important updates from the hospital or medical staff. Notifications can also be sent when there is an important change, such as when the mother enters the delivery room, when the baby is born, or when the mother is moved to recovery.

This operation tracking system is important because many hospitals do not currently provide a simple way for families to stay informed during labor. In most cases, relatives wait outside without receiving enough information, which can increase stress and worry. By providing live updates and clear information, the system can help families feel calmer and more connected to the process. It can also improve communication between hospital staff and family members because information can be updated in one place instead of being repeated many times.

In addition to labor monitoring, the system also includes pregnancy tracking features for expecting mothers. This part of the platform is designed to help users monitor their pregnancy journey from the early weeks until childbirth. Pregnant women will be able to create an account, log in securely, and enter personal information such as name, age, due date, and last menstrual period. Based on this information, the system can automatically calculate the current pregnancy week and trimester.
The pregnancy tracking environment will also allow users to record weekly information such as weight changes, symptoms, doctor notes, and personal notes. Mothers can monitor how their pregnancy changes over time and keep all important information stored in one place. The system can also show baby growth and development information for each week of pregnancy. For example, users can learn about the baby’s size, body development, and important changes during each trimester. This makes the platform useful not only during labor, but also throughout the entire pregnancy process.
Another important feature of the system is the personalized dashboard. After logging in, each user will be able to access their own dashboard where they can see all pregnancy records, current week, trimester, due date, and recent updates. Family members who are connected to the account may also be able to access operation updates during labor. The dashboard helps keep all information organized and easy to understand.

The system will also include a secure login and registration process. Users will need to create an account before accessing the platform. This ensures that only authorized people can view sensitive pregnancy or labor information. Secure session management will also be included so that user data remains protected. Error handling and validation will be used to avoid incorrect information and duplicate records. For example, the system can prevent users from entering the same pregnancy week twice.

From a technical perspective, the application will be developed using PHP, MySQL, HTML, CSS, JavaScript, and XAMPP. PHP will be used for the backend logic and connection with the database, while MySQL will store user information, pregnancy records, labor updates, and notifications. HTML and CSS will be used to design the interface and make it easy to use. JavaScript can also be included for notifications, alerts, and real-time updates.

The system is expected to be simple, modern, and user-friendly. Since not all users are familiar with technology, the design should be easy to understand and navigate. Clear menus, simple forms, and organized dashboards will help users interact with the platform without confusion. The goal is to create a system that can be used by pregnant women, family members, and even hospital staff in a practical way.

Overall, the proposed solution aims to create a complete support platform that combines pregnancy monitoring with labor tracking. The main priority is to improve communication during childbirth and reduce stress for families. At the same time, the pregnancy tracking features make the system useful before labor as well. By combining these two functions into one application, the platform can provide better support, organization, and peace of mind during one of the most important moments in a family’s life.

The main aim of the project is to create a system focused on operation tracking during labor while also supporting pregnancy monitoring.

Application Description
mySQL, HTML, CSS, Php, and XAMPP. It is designed to provide two main environments: one for pregnancy tracking and one for operation monitoring. The pregnancy tracking environment allows users to register, log in, and store important information such as due date, age, symptoms, weight, trimester, and pregnancy week. Users can also add weekly notes and monitor their overall pregnancy progress through a personalized dashboard.
The second environment focuses on operation monitoring during labor and delivery. Family members can use this section to receive updates about the childbirth process, operation stages, and important notifications from medical staff. This feature helps families remain informed even if they are not physically present at the hospital.
The system also includes important features such as secure login, database storage, session management, duplicate entry prevention, trimester calculation, baby development information, and notifications. The interface is designed to be simple, modern, and easy to use for all users.
